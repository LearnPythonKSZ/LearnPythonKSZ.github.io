Was bedeutet Programmieren ?

Dieser Kurs soll euch bei dem Einstieg in das Programmieren, mithilfe der Programmiersprache Python, erleichtern. Doch, was bedeutet eigentlich "Programmieren"? Einfach gesagt bedeutet programmieren das Erstellen von Software. Dies funktioniert über eine Programmiersprache, in unserem Fall Python. Computer können nämlich nur mit sogenanntem Maschinensprache/Binärcode arbeiten. Also "übersetzt" Python die verschiedenen Befehle in Binärcode, welcher dann der Computer (genauer gesagt der Prozessor) ausführen kann. Doch leider versteht Python nicht, wenn man ihm sagt: "was ist 1+1?". Python ist nämlich eine eigene Sprache, mit eigenen Befehlen, welche allerdings sehr offensichtlich und leicht verständlich sind, wenn man grundlegende englische Begriffe versteht. Ein Beispiel dafür wäre die "If" Bedingung. Die deutsche Übersetzung des Wortes "If" ist "Wenn". Bei einer "If" Anweisung überprüft Python eine Bedingung. Wenn diese Wahr ist wird eine bestimmte Anweisung ausgeführt, sonst eine andere. Also stimmmt die triviale deutsche Übersetzung stark mit der Funktionsweise überein. Zu den "If-Statements" kommen wir allerdings erst später.

Achtung!

Wenn du einen Begriff nicht verstehst, schaue am Ende der Seite nach. Dort werden die schwierigen Worte noch einmal erklärt.

Welche Programmiersprachen gibt es?

Es gibt tausende verschiedene Programmiersprachen, welche alle ihre eigenen Vor- und Nachteile haben. Zu den meist verbreiteten Programmiersprachen zählen die Programmiersprache C, beziehungsweise ihre Weiterentwicklungen C++ und C#, Java und auch Python. sind die meist verbreiteten Programmiersprachen und so ist zum Beispiel ein Großteil von Windows und dem Mac OS X Betriebssystem in diesen Programmiersprachen geschrieben. Java wird vor allem im Internet und Multimediaanwendungen eingesetzt. Sie ist auch die verbreiteste Programmiersprache für Mobilgeräte. Python ist so verbreitet, weil sein Anwendungsspektrum sehr breit ist und es für Anfänger sehr leicht verständlich ist. Ausserdem ist die Anwendung und der Quellcode kostenlos erhältlich, deshalb ist sie auch auf fast allen Betriebssystemen vorhanden.

Begriffserklärungen

Software: Der Teil des Computers den man nicht anfassen kann. Software (syn. Programm) kann von einem Computer ausgeführt werden und sagt bestimmter Hardware (Sachen die man anfassen kann) was sie wie zu erledigen bzw. zu berechnen hat.

Binärcode/Maschinensprache: Maschinensprache besteht aus zwei "Zeichen" 1 und 0. Dies da der Computer nur 2 Zustände kennt: Strom fliesst (1), beziehungsweise kein Strom fliesst (0). Diese Maschinensprache kann dann von einem Computer ausgeführt werden.

Quellcode: Ein Text, welcher in einer Programmiersprache geschrieben worden ist und von Menschen gelesen werden kann. Der Computer kann die verschiedenen Befehle einer Programmiersprache in sogenannter Maschinensprache übersetzten, welche dann von einem Computer ausgeführt werden kann.
